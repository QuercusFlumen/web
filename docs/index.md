Bosque, agua, microbiología y evolución ecológica aplicada a sistemas acuáticos naturalistas.

---

# Introducción

Quercus Flumen nace de la observación de una relación fundamental:

- materia vegetal,
- agua,
- microbiología,
- tiempo,
- organismos vivos,
- y evolución ecológica.

El proyecto no interpreta el blackwater como simple coloración del agua.

Lo interpreta como un sistema dinámico donde:
- el bosque transforma materia vegetal,
- el agua actúa como medio de transferencia,
- la microbiología reorganiza el sistema,
- y los organismos vivos responden dentro de un entorno en evolución.

Wild es la herramienta principal desarrollada dentro de esta filosofía.

Pero Quercus Flumen no se limita a Wild.

El objetivo de esta documentación es construir una base clara para comprender:
- cómo se transforma la materia vegetal,
- cómo se prepara y utiliza Wild,
- cómo evoluciona el agua,
- cómo se interpretan los límites del sistema,
- y cómo peces, gambas, plantas, biofilm y biotopos forman parte de ecosistemas acuáticos naturalistas.

# Qué es Wild dentro de Quercus Flumen

Wild es un concentrado botánico forestal seco y molido desarrollado a partir de distintas fases vegetales de *Quercus pyrenaica*.

Su función no es simplemente oscurecer el agua.

Wild se desarrolla como:
- herramienta de extracción,
- base vegetal compleja,
- sistema de interacción orgánica,
- y punto de partida para construir biotopos naturalistas, blackwater y sistemas ricos en evolución microbiológica.

La mezcla de fases vegetales permite trabajar con perfiles complementarios:
- fracciones más solubles,
- fracciones más estructurales,
- comportamientos distintos de decantación,
- y dinámicas diferentes en el agua.

---

# Qué no pretende ser este proyecto

Quercus Flumen no pretende ofrecer:
- recetas universales,
- promesas absolutas,
- protocolos rígidos,
- ni afirmaciones simplificadas sobre peces, plantas o gambas.

La documentación trabaja desde:
- observación,
- prudencia,
- compatibilidad ecológica,
- interpretación del sistema,
- y comprensión progresiva del biotopo.

No se afirma que Wild mejore directamente una especie concreta.

Se analiza cómo determinados sistemas ricos en materia vegetal, aguas blandas, biofilm, refugio, microbiología y estabilidad pueden resultar coherentes con ciertos organismos y biotopos.

# Filosofía general

Quercus Flumen parte de una idea sencilla:

El agua no es solo agua.

En un sistema naturalista:
- transporta materia,
- reorganiza compuestos,
- permite interacción microbiológica,
- refleja estabilidad,
- y conecta bosque, organismos vivos y tiempo.

Wild nace de esa relación entre:
- *Quercus pyrenaica*,
- transformación vegetal,
- extracción,
- agua blanda,
- microbiología,
- y sistemas acuáticos vivos.

La intención no es copiar literalmente la naturaleza.

La intención es aprender de sus dinámicas para construir sistemas acuáticos más coherentes, observables y ecológicamente ricos.
